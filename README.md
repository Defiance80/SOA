## Enhancement Philosophy

**Standards Applied:**
1. **No lazy lines** - poetic noun craft, deliberate word choice
2. **Sensory immersion** - always grounded in physical experience
3. **Cultural authenticity** - Black American humor (timing > slang)
4. **Mature tone** - romance allusive, violence consequential
5. **Flow paramount** - reader doesn't notice prose, only world
6. **Canon-accurate** - Bible (locked facts) takes precedence over legacy manuscript

**Canon Hierarchy (per Writing Bible):**
1. Writing Bible §2 (locked rulings) - **ABSOLUTE**
2. Live project docs (Overview, Story Arch)
3. Illustrated Chapters 1-10 - current prose truth
4. novel.md (original 18 chapters) - reference when not conflicting
